### 3. **学生功能**

#### 3.1 **查询课程 API**

##### 3.1.1 **查看课程信息**(分页查看)

###### 基本信息：

**接口地址**: `/student/get_courses`

**请求方法**: `GET`

###### 请求参数:

参数格式：queryString

参数说明：

| 参数名称 | 是否必须 | 示例 | 备注                                       |
| -------- | -------- | ---- | ------------------------------------------ |
| page     | 是       | 1    | 分页查询的页码，如果未指定，默认为1        |
| pageSize | 是       | 20   | 分页查询的每页记录数，如果未指定，默认为20 |

请求数据样例：

```shell
/student/get_courses?page=1&pageSize=20
```

###### 响应数据:

参数格式：application/json

参数说明：

| 参数名           | 类型   | 是否必须 | 说明                             |
| ---------------- | ------ | -------- | -------------------------------- |
| code             | int    | 必须     | 状态码，200表示成功，400表示失败 |
| message          | string | 非必须   | 响应消息                         |
| total            | int    | 必须     | 总记录条数                       |
| data             | array  | 非必须   | 课程信息列表                     |
| \|- course_id    | number |          | 课程ID                           |
| \|- course_no    | string |          | 课程号                           |
| \|- course_name  | string |          | 课程名称                         |
| \|- credit       | double |          | 学分                             |
| \|- teacher_name | string |          | 授课老师                         |
|                  |        |          |                                  |
|                  |        |          |                                  |

**示例响应（成功）**:

```json
{
    "code": 200,
    "message": "课程查询成功",
    "total":2,
    "data": [
        {
            "course_id": 1,
            "course_no": 101,
            "course_name": "数据结构",
            "credit": 2,
            "teacher_name": "张翀岳",

        },
        {
            "course_id": 2,
            "course_no": 102,
            "course_name": "操作系统",
            "credit": 3,
            "teacher_name": "乔文瑛",
        }
    ]
}
```

**示例响应（失败）**:

```json
{
    "code": 400,
    "message": "课程查询失败",
    "data":null
}
```

#### 3.4 **查看学生课表 API**

##### 基本信息

**接口地址**: `/student/get_timetable/{student_id}`

**请求方法**: `GET`

##### 请求参数:

| 参数名     | 类型 | 是否必填 | 说明   |
| ---------- | ---- | -------- | ------ |
| student_id | int  | 是       | 学生ID |

##### 响应数据:

参数格式：`application/json`

| 参数名               | 类型   | 是否必须 | 说明                             |
| -------------------- | ------ | -------- | -------------------------------- |
| code                 | int    | 必须     | 状态码，200表示成功，400表示失败 |
| message              | string | 非必须   | 响应消息                         |
| data                 | object | 非必须   | 返回的课表信息列表               |
| \|- day_of_week      | string |          | 上课的星期几（周一、周二等）     |
| \|- courses          | object |          | 当前日期的课程信息列表           |
| \|- \|- course_no    | string |          | 课程号                           |
| \|- \|- course_name  | string |          | 课程名称                         |
| \|- \|- credit       | double |          | 学分                             |
| \|- \|- teacher_name | string |          | 授课教师姓名                     |
| \|- \|- time_slot    | string |          | 上课时间（xx:xx-xx:xx）          |

> 上课地点？

**示例响应（成功）**:

```json
{
    "code": 200,
    "message": "课表查询成功",
    "data": [
        {
            "day_of_week": "周一",
            "courses": [
                {
                    "course_no": "101",
                    "course_name": "数据结构",
                    "credit": 2,
                    "teacher_name": "张翀岳",
                    "time_slot": "08:30-10:30"
                },
                {
                    "course_no": "102",
                    "course_name": "操作系统",
                    "credit": 2,
                    "teacher_name": "乔文瑛",
                    "time_slot": "13:30-15:30"
                }
            ]
        },
        {
            "day_of_week": "周三",
            "courses": [
                {
                    "course_no": "103",
                    "course_name": "计算机网络",
                    "credit": 3,
                    "teacher_name": "张树刚",
                    "time_slot": "08:30-10:30"
                }
            ]
        }
    ]
}
```

**示例响应（失败）**:

```json
{
    "code": 400,
```
