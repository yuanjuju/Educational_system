

#### 1. 学生表

| 属性名           | 数据类型         | 说明             |
| ---------------- | ---------------- | ---------------- |
| student_id       | int              | 自增主键         |
| student_no       | string           | 学号 账号 unique |
| student_name     | string           | 姓名             |
| student_dept_id  | int              | 外键，院系id     |
| student_password | string           | 密码             |
| student_DOB      | string           | 生日（非必须）   |
| student_sex      | enum("男"，"女") | 性别（非必须）   |



#### 2. 教师表

| 属性名                   | 数据类型                        | 说明             |
| ------------------------ | ------------------------------- | ---------------- |
| teacher_id               | int                             | 自增主键         |
| teacher_no               | string                          | 工号 账号 unique |
| teacher_name             | string                          | 姓名             |
| teacher_dept_id          | int                             | 外键，院系id     |
| teacher_password         | string                          | 密码             |
| （可选）teacher_position | enum ('教授', '副教授', '讲师') | 职位             |



#### 3. 管理员表

| 属性名         | 数据类型 | 说明             |
| -------------- | -------- | ---------------- |
| admin_id       | int      | 自增主键         |
| admin_no       | string   | 工号 账号 unique |
| admin_name     | string   | 姓名             |
| admin_password | string   | 密码             |

> 这里为了区分语义添加了前缀，实际设计时可以把前缀删去
>
> 是否将三种角色建立在一张用户表上，以属性role来区分，就是每次操作都要判断一下角色role有点麻烦



#### 4. 课程表

| 属性名      | 数据类型 | 说明                                                         |
| ----------- | -------- | ------------------------------------------------------------ |
| course_id   | int      | 自增主键                                                     |
| course_no   | string   | 课程号 unique                                                |
| course_name | string   | 课程名                                                       |
| credit      | float    | 学分                                                         |
| day_of_week | string   | 上课时间（周x）                                              |
| time_slot   | string   | 上课时间（x点-x点）                                          |
| teacher_id  | int      | 外键 来自教师表 表示授课教师<br />约束：同一个授课教师的上课时间不能冲突 |

#### 5. 选课表

| 属性名       | 数据类型 | 说明                                                         |
| ------------ | -------- | ------------------------------------------------------------ |
| student_id   | int      | 主键之一，外键来自学生表<br />约束：同一学生选择的课程不能冲突 |
| course_id    | int      | 主键之一，外键来自课程表                                     |
| buy_textbook | bool     | 表示是否购买教材                                             |
| grade        | float    | 成绩<br />约束：0-100                                        |

#### 6. 教学反馈表

| 属性名     | 数据类型 | 说明                           |
| ---------- | -------- | ------------------------------ |
| student_id | int      | 主键之一，外键来自学生表       |
| course_id  | int      | 主键之一，外键来自课程表       |
| feedback   | string   | 反馈内容，教师端要能够查看反馈 |

#### 7. 学院表

| 属性名    | 数据类型 | 说明     |
| --------- | -------- | -------- |
| dept_id   | int      | 自增主键 |
| dept_name | string   | 学院名   |

> 额外的：忘记密码，通过邮箱认证找回（在每种身份里面加一个xxx_email）
