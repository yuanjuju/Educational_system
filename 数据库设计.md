

#### 1. department 表：存储学院信息

| 属性名   | 数据类型     | 说明   | 备注     |
| -------- | ------------ | ------ | -------- |
| DeptID   | int          | 学院id | 自增主键 |
| DeptName | varchar(100) | 学院名 | 唯一     |



#### 2. student 表：存储学生信息

| 属性名     | 数据类型     | 说明     | 备注                 |
| ---------- | ------------ | -------- | -------------------- |
| StudentID  | int          | 自增主键 |                      |
| student_no | varchar(20)  | 学号     | 唯一                 |
| Name       | varchar(100) | 姓名     | 非空                 |
| Gender     | char(1)      | 性别     | 1表示男生，2表示女生 |
| DOB        | date         | 出生日期 | 非空                 |
| DeptID     | int          | 学院id   | 外键来自department表 |
| Username   | varchar(50)  | 用户名   | 唯一                 |
| Password   | varchar(50)  | 密码     | 非空（默认为学号）   |



#### 3. teacher 表：存储教师信息

| 属性名     | 数据类型     | 说明     | 备注                            |
| ---------- | ------------ | -------- | ------------------------------- |
| TeacherID  | int          | 自增主键 |                                 |
| teacher_no | varchar(50)  | 工号     | 唯一                            |
| Name       | varchar(100) | 姓名     | 非空                            |
| Gender     | char(1)      | 性别     | 1表示男生，2表示女生            |
| Title      | varchar(50)  | 职称     | enum ('教授', '副教授', '讲师') |
| DeptID     | int          | 学院id   | 外键来自department表            |
| password   | varchar(100) | 密码     | 非空                            |



#### 4. admin 表：存储管理员信息

| 属性名   | 数据类型     | 说明 | 备注 |
| -------- | ------------ | ---- | ---- |
| username | varchar(255) | 主键 | 唯一 |
| admin_no | varchar(255) | 工号 | 唯一 |
| password | varchar(255) | 密码 | 非空 |



#### 5. course 表：存储课程信息

| 属性名     | 数据类型     | 说明                | 备注                                                         |
| ---------- | ------------ | ------------------- | ------------------------------------------------------------ |
| CourseID   | int          | 自增主键            |                                                              |
| course_no  | varchar(50)  | 课程号              | 非空 唯一                                                    |
| CourseName | varchar(100) | 课程名              | 非空                                                         |
| Credits    | float        | 学分                | 约束：大于0                                                  |
| TeacherID  | int          | 授课教师号          | 外键：来自teacher表（体现教师和课程的联系）约束：同一教师的上课时间不能冲突 |
| DayOfWeek  | varchar(10)  | 上课时间（周x）     |                                                              |
| TimePeriod | varchar(20)  | 上课时间（x点-x点） |                                                              |



#### 6. sc 表：存储学生选课信息（体现学生和课程的联系）

| 属性名    | 数据类型      | 说明                     | 备注 |
| --------- | ------------- | ------------------------ | ---- |
| StudentID | int           | 主键之一，外键来自学生表 |      |
| CourseID  | int           | 主键之一，外键来自课程表 |      |
| Grade     | decimal(5,2)  | 成绩                     |      |
| BuyBook   | int           | 表示是否购买教材         |      |
| feedback  | varchar(1000) | 学生对课程的反馈         |      |
